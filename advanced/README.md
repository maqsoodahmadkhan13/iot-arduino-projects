# 🔴 Advanced Projects

Full IoT builds: MQTT/HTTP to cloud services or dashboards (e.g. ThingSpeak, Home Assistant, Firebase), multi-sensor data logging, automation logic, OTA updates, power optimization for battery-powered nodes.

**Prerequisites:** Comfortable with intermediate projects, basic Wi-Fi/MQTT/HTTP concepts, an ESP32/ESP8266 board.
